⚡ FlowForge

FlowForge is a high-performance, dark-themed time-tracking web application meticulously designed for deep work and productivity analytics. It combines a sleek, modern UI with robust local persistence to offer a seamless productivity experience.

1. Core Features

Time Tracking: Effortlessly log your activities with integrated task and tag selection.


Focus Sessions: A dedicated Pomodoro-style circular countdown timer with custom presets (25m, 45m, 60m, 90m) to manage deep work blocks.


Dynamic Analytics: Visualize your productivity trends through interactive, color-coded stacked bar charts.


Task & Tag Management: Full organization of your workflow with custom task categories and tag-based filtering.


Always-on-Top Timer: A floating mini-timer window powered by the Document Picture-in-Picture API, keeping your countdown visible across all tabs and minimized states.

Data Privacy & Persistence: Your data stays yours. All tracking history and settings are stored locally in your browser.

2. Tech Stack

Framework: Next.js (App Router) 


Styling: Tailwind CSS 


Visualizations: Recharts 


Iconography: Lucide React 


State Management: React Context API 

3. Getting Started
Prerequisites:

Node.js (v18 or later)

npm or yarn

4. Installation:

a) Clone the repository:

Bash
git clone https://github.com/YOUR_USERNAME/flowforge.git

b)Install dependencies:

Bash
npm install

5. Development:
Run the development server locally:

Bash
npm run dev
Open http://localhost:3000 in your browser to view the application.

6. Deployment
This project is optimized for deployment on Vercel and is configured as a Progressive Web App (PWA) for easy installation on Windows, macOS, and mobile devices.
